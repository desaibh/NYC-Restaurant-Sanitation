# PROJECT 2 - NYC Restaurant Health Grades

## PROJECT MVP
Build a restaurant health grades application that allows users to login/register and submit comments about an experience that they had at the restaurant.

## TECHNOLOGIES
* React
* Firebase
* Open Data
* HTML / CSS
